# Pomodoro Timer (β)

本プロジェクトは、個人の生産性向上を支援するためのポモドーロ・タイマーです。ブラウザ上で動作するように設計されており、特別なインストールを必要とせず、オフライン環境でも利用可能です。

[アプリケーションを実行する](https://momerio.github.io/PomodoroTimer/)

---

## 主な機能

- **タイマーのカスタマイズ**: 作業時間、小休憩、大休憩を秒単位で設定。
- **ブラウンノイズ生成**: Web Audio APIを用いたノイズ生成機能を搭載。外部ファイル不要で集中環境を構築。
- **セッション管理**: 作業内容の入力とログの自動記録。
- **外部出力機能**: 蓄積された作業ログをCSV形式で保存可能。
- **デザイン設定**: ライトモードとダークモードの切り替え。
- **データの保存**: 設定やログはブラウザのLocalStorageに保存され、再起動後も保持されます。
- **入力値の直接編集**: タイマーの残り時間やログの項目をダブルクリックで編集可能。

## 技術仕様

- **フロントエンド**: HTML5, CSS3, JavaScript (Vanilla JS)
- **音声処理**: Web Audio API
- **デザイン資産**: Google Fonts (Inter), Font Awesome
